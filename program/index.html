Program

あなたのペースで、
自分を取り戻していくために。

Pale Hourのプログラムは、無料の診断から静かに始まります。
どこまで進むかは、いつでもあなたが選べます。急がせることはありません。

Journey

Pale Hourでの歩み方

それぞれの段階は「次を買わせるための階段」ではなく、自分への理解が深まっていくプロセスとして設計しています。

  1. Start

    InstagramやThreadsで出会う

    日々の発信の中で、「これは私のことかもしれない」と感じたら。

  2. Step 01 — 無料

    恋愛フェーズ診断

    6つの質問で、今の恋愛の状態を「夜から朝へ」の5段階で見つめます。Midnight / First Light / Turning Tide / Sunrise / Morning Tide。

  3. Step 02 — 無料

    LINE登録・診断結果の受け取り

    公式LINEで診断結果と、あなたのフェーズに合わせた最初の一歩をお届けします。

  4. Step 03 — 無料

    LINE限定パターン診断・タイプ別無料Note

    恋愛で繰り返しやすいパターンをさらに深く見つめ、タイプごとの無料Noteで「なぜ苦しくなるのか」の仕組みを言葉にします。

  5. Step 04 — 有料Note

    恋愛の教科書

    恋愛で自分を見失う仕組みと、自分軸へ戻る手順を体系的にまとめた一冊。読み終えたあと、購入者特典として無料の個別相談をご案内します。

  6. Step 05 — 不定期開催

    グループコンサル

    少人数で、恋愛のその先——境界線・自分の軸・これからの選択——を整えていく時間。開催時期・内容はLINEでご案内します。
